# Joint Venture Dashboard

Open `index.html` in any modern browser to use the dashboard.

To make it accessible to anyone anywhere, upload `index.html` to a static hosting service such as SharePoint, OneDrive web hosting, GitHub Pages, Netlify, Vercel, or your company website hosting. The file has no external dependencies.

New entries, status updates, G.M. actions, G.M. owner, notes edits, and deleted records are saved permanently in the browser that makes them. Use **Export CSV** when you need a copy of the latest list.

Delete password: `admin123`

If you need every user to update the same live master data at the same time, the next step is a hosted database-backed version instead of a static HTML file.
